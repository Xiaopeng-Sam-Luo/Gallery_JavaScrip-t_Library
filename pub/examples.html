Create Gallery

Step1

Select the div in which you want to put the gallery in.

<div id='myId1'> </div>

Step2

Decide the number of images you want to list in the gallery.

const test = gallery('#myId')

test.makeholders(4)

You just create a gallery with four empty position

Step3

Add images to the gallery

test.insertimages(i, url)

Where i is the index of the gallery positoin and url is the address of the image you want to upload

If the postion you choice already has a image, then the bew image would replace the old iamge

After insertion each image can be view in fullscreen by clicking on it.

Drag, reorder $ resize

You can allow interaction such as drag and rezise with imges by doing the following:

test.drag()

test.order()

test.changesize(0, 'width:20px')

After all above procedures you get yourself a intreactable art gallery !
